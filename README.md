# Cafeteria Pre-Ordering System

A web-based application that allows users to browse menus, place pre-orders, and pick up food using QR codes.

## Features

* User login & registration
* Menu browsing
* Cart & pre-order
* QR-based pickup
* Admin order management

## Tech Stack

* HTML, CSS, JavaScript
* Node.js, Express
* MongoDB

## Run Locally

```bash
npm install
node server.js
```
